번트(BUNT) 오버나잇 엔진 v2.0
오늘 상한가 종목을 장 마감에 사서, 내일 아침 시가에 파는 자동매매 엔진
주식이 하루에 +29% 이상(상한가 근접) 폭등하면, 뉴스와 SNS를 타고 관심이 퍼집니다.
"나도 내일 사야지!" 하는 사람들이 밤새 쌓이고,
다음날 아침 장이 열리면 매수 주문이 몰리면서 시가가 어제 종가보다 높게 형성됩니다.
이 오버나잇 갭(하룻밤 사이 가격 상승)을 매일 자동으로 반복 수확하는 것이 번트의 원리입니다.

번트는 감정 없이 매일 자동으로 양의 기대값 게임을 반복합니다. 카지노의 하우스 엣지와 같은 원리입니다.
매매 파이프라인
[1단계] 종목 스캔 (1차 매수 크론 시작 시 자동 실행)
  │  네이버 모바일 API → 코스닥 전종목 (~1,820개) 등락률 조회
  │  등락률 상위 30종목 → 개별 OHLCV 상세 조회
  ▼
[2단계] 7단계 필터링
  │
  │  ① 신규상장 제외    등락률 > 30% (가격제한폭 밖)
  │  ② 관리종목 제외    거래소 관리종목/정리매매 블랙리스트
  │  ③ 고정 상한가 제외  시가=고가=종가 (종일 고정, 체결 불가)
  │  ④ 등락률 필터      ≥ 29% (상한가 근접만)
  │  ⑤ 종가=고가 필터   종가/고가 = 100% (장마감까지 매수세 유지)
  │  ⑥ 연속 재진입 필터  전일 매수 종목 제외
  │  ⑦ DART 악재 필터  연속적자(3기중 2기+) / 악재공시(90일) 차단
  │
  │  → 최대 10종목 선정 (등락률 내림차순)
  ▼
[3단계] 분할 매수 (50/50)
  │
  │  1차 15:05~06  스캔 → 필터 → 정규장 시장가 주문 (50%)
  │  2차 15:21~22  1차 선정 종목 장마감 동시호가 주문 (50%)
  │  예수금 ÷ 종목수 = 균등 배분
  ▼
[4단계] 오버나잇 보유
  │  사람들이 "내일 사야지!" 하는 동안 기다림
  ▼
[5단계] 익일 매도
  │
  │  실전: 08:50 시장가 주문 → 09:00 동시호가(시가) 체결
  │  모의: 09:01 시가(open) 조회 → 시가로 정산
  │  → 밤새 쌓인 매수세로 시가 > 종가 = 수익
  │
  │  매도 안전장치 (실전):
  │  실패 시 즉시 재시도(2회)
  │  → 프로그레시브 리트라이 (30초~10분, 5단계)
  │  → 최종 실패 시 이메일 알림
  ▼
[6단계] 정산
  수수료 0% (국내수수료우대) + 매도세 0.20%
  = 왕복 0.20% (실전: KIS API에서 실제 비용 자동 조회)
  끝. 매일 저녁 자동 반복.
DART 악재 필터 (2단계 ⑦)
DART Open API로 매수 전 종목별 재무 리스크를 자동 체크합니다.
